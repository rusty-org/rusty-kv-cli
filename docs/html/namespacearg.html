Parses and validates CLI arguments into connection info.

Parses command-line options into KvConnectionInfo.

  • Sets defaults: localhost:6379, no auth.
  • On <2 args, prints usage and exits.
  • Handles flags: -p, -h, -U, -P, -url.
  • Constructs info.url if not provided.
Parameters
argcArg count.
argvArg values.
infoKvConnectionInfo to configure.

Supports -p, -h, -U, -P, and -url flags.

Exits on missing required values or invalid URI.

Parameters
argcNumber of CLI args.
argvArray of arg strings.
infoOutput connection info to populate.
