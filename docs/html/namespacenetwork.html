Parses a KV connection URI into host, port, and optional credentials.

Parses a connection URI into its components.

Uses regex to extract user, password, host, and port. Converts "localhost" to "127.0.0.1".

Parameters
uriConnection URI string.
infoKvConnectionInfo to populate.
Returns
True if URI matched expected pattern; false otherwise.

Supported formats:

  • kv://user:password@host:port
  • kv://host:port
  • kv://user:password@host
Parameters
uriURI string to parse.
infoOutput connection info struct to populate.
Returns
True on successful parse; false otherwise.
